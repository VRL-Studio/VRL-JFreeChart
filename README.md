VRL-JFreeChart
==============

Integrates JFreechart to VRL

## Dependencies

- Java >= 1.6 
- [NetBeans](http://netbeans.org) >= 7.01 
- [VRL-Studio](http://vrl-studio.mihosoft.eu) >= 0.4.4.3
- [VRL](https://github.com/miho/VRL) >= 0.4.2.6
